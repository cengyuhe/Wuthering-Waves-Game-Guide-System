# Wuthering Waves Guide System

## Project Description

This project is a web-based guide system for the game Wuthering Waves.
It provides character guides, weapon guides, echo guides, team recommendations, user authentication, and favourite management.

------------------------------------------------

Software Requirement

Operating System
- Windows 10 / Windows 11

Web Server
- XAMPP 8.2.12

PHP
- PHP 8.2

Database
- MariaDB 10.4

Browser
- Microsoft Edge

------------------------------------------------

Installation Guide

1. Install XAMPP

https://www.apachefriends.org/

2. Start Apache and MySQL.

3. Copy the project folder into:

xampp/htdocs/

4. Open phpMyAdmin.

5. Create a database:

characters

6. Import:

wuthering_waves_db.sql

7. Open browser

http://localhost/Wuthering-Waves-Guide-System/

------------------------------------------------
